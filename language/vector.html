R Primer

Creating a vector

Vectors are variables with one or more values of the same type: logical, integer, real, complex, string (or character) or raw.

The simplest way to create vectors is to use the function c() which is a generic function that combines its arguments into vectors --- in facts, c() may be used to combine arguments of different types into more complicated structures; we will see this later.

For instance, to create a vector with the following mathematical representation

we enter the following command

x <- c(.1, .2, .3)

Another common way to construct vectors is to use sequence. Suppose we want to create the following vector

This vector can be easily created by

x <- 1:10

Here, 1:10 is the notation to create a sequence of number from 1 to 10. So, to create a vector the contains all integer from 1 to 100 we simply execute

x <- 1:100

The 1:10 notation to construct a vector is useful, but sometime we need to construct vector whose elements can be specified as more complicated sequences. For instance

To use create this vector we have to use to a function seq which take as first argument the starting value of the sequence, as second argument the final value of the sequence, and as third argument the size of the increment:

x <- seq(1, 10, .5)

We will see, when we discuss the object oriented nature of R, that a vector is an object of class numeric. This means that every vector has some attributes that is common to all the vectors, regardless of the actual content of the vector.

One of the most important attributes of a vector is its length, that is, the number of element it contains. To get the size of the vector we use length

length(x)

## [1] 19

Here, we get 19 because the last values we assigned to x where the terms of the sequence specified by seq(1, 10, .5).

Getting elements of a vector

Element of a vector can be extracted by using index, in a very similar way we do with the linear algebra notation. The third element of the vector can be obtained by 

x[3]

## [1] 2

We can extract more element at time by using, guess what, vector of indexes. For example, to extract the first 5 element of the vector

x[1:5]

## [1] 1.0 1.5 2.0 2.5 3.0

But we can also extract the evel element of the vector x, the elements [1, 3, 5, 7, 9] by using

x[seq(1, 10, 2)]

## [1] 1 2 3 4 5

More generally, we can extract the third and ninth elements like this

x[c(3,9)]

## [1] 2 5

Notice that we can index by other vectors. So, for instance, 

idx  <- seq(1,10,2)
x[idx]

extracts the same elements as x[seq(1,10,2)].
